XelToFab converts topology optimization density fields into clean, fabrication-ready triangle meshes. It handles the full post-processing pipeline: preprocessing, mesh extraction, smoothing, and quality analysis.

## What it does

Given a density field (a numpy array with values in \[0, 1\]) from any topology optimization solver, XelToFab:

1. **Preprocesses** the field — Gaussian smoothing, thresholding, morphological cleanup
2. **Extracts** geometry — marching cubes (3D → triangle mesh) or marching squares (2D → contours)
3. **Smooths** the result — Taubin filtering to remove staircase artifacts

## Key features

- **Multi-format input** — NumPy, MATLAB, VTK, HDF5, CSV
- **2D and 3D** — contour extraction or triangle mesh generation
- **SDF support** — direct extraction from signed distance fields (neural models)
- **Quality metrics** — aspect ratio, min angle, scaled Jacobian via PyVista
- **CLI and Python API** — `xtf` command-line tool or import directly

## Quick example

```python
from xeltofab.io import load_density, save_mesh
from xeltofab.pipeline import process

result = process(load_density("density_field.npy"))
save_mesh(result, "output.stl")
```
