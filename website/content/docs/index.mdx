XelToFab converts scalar design fields into clean, fabrication-ready triangle meshes. It handles the full post-processing pipeline: preprocessing, mesh extraction, smoothing, repair, and quality analysis.

<img src="/images/getting-started/hero-overview.png" alt="XelToFab converts density fields into clean triangle meshes" width={800} />

## What it does

Given a scalar field (a numpy array) from a topology optimization solver, neural network, or other computational design tool, XelToFab:

1. **Preprocesses** the field — Gaussian smoothing, thresholding, morphological cleanup
2. **Extracts** geometry — marching cubes (3D → triangle mesh) or marching squares (2D → contours)
3. **Smooths** the result — Taubin filtering to remove staircase artifacts

## Key features

- **Multi-format input** — NumPy, MATLAB, VTK, HDF5, CSV
- **2D and 3D** — contour extraction or triangle mesh generation
- **Multiple field types** — density fields, SDFs, occupancy fields, and neural field outputs
- **Quality metrics** — aspect ratio, min angle, scaled Jacobian via PyVista
- **CLI and Python API** — `xtf` command-line tool or import directly

## Quick example

```python
from xeltofab.io import load_field, save_mesh
from xeltofab.pipeline import process

result = process(load_field("field.npy"))
save_mesh(result, "output.stl")
```
