## Requirements

- **Python 3.13+**
- **[uv](https://docs.astral.sh/uv/)** (recommended) or pip for installation

## Install from PyPI (recommended)

The stable release is available on [PyPI](https://pypi.org/project/xeltofab/):

```bash
pip install xeltofab
```

With optional extras:

```bash
pip install "xeltofab[vtk,hdf5]"
```

Or using uv:

```bash
uv add xeltofab
```

## Install from source (development)

For the latest development version or to contribute:

```bash
git clone https://github.com/xarthurx/XelToFab.git
cd XelToFab
uv sync
```

This installs all core dependencies: `numpy`, `scipy`, `scikit-image`, `trimesh`, `pydantic`, `matplotlib`, and `click`.

You can also install the development version directly from GitHub:

```bash
pip install git+https://github.com/xarthurx/XelToFab.git
```

Or with uv:

```bash
uv add git+https://github.com/xarthurx/XelToFab.git
```

## Optional format support

Some input formats require additional libraries. Install everything at once:

```bash
uv sync --all-extras
```

Or install individual extras selectively:

| Extra | Command | Adds support for |
|-------|---------|------------------|
| `vtk` | `uv sync --extra vtk` | `.vtk`, `.vtr`, `.vti` files (via PyVista) |
| `hdf5` | `uv sync --extra hdf5` | `.h5`, `.hdf5`, `.xdmf` files (via h5py) |
| `all-formats` | `uv sync --extra all-formats` | All optional formats |

## Verify installation

Check that the CLI is available and see which formats are ready:

```bash
uv run xtf --help
```

List all supported input formats and their availability status:

```bash
uv run xtf formats
```

See [xtf formats](/docs/cli/formats) for the full list of supported formats and their dependencies.

## Running tests

```bash
uv run pytest tests/ -v
```
